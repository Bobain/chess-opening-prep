## Créer une nouvelle étude

### Étapes
1. Aller sur https://lichess.org/study [À CONFIRMER]
2. Cliquer sur "+ Create a study" (ou "Créer une étude") [À CONFIRMER]
3. Donner un nom (ex: "Blancs - Gambit Dame") [À CONFIRMER]
4. Choisir la visibilité : Unlisted ou Private [À CONFIRMER — quelles options ?]
5. Cliquer sur "Start" / "Créer" [À CONFIRMER]

### Notes
- Créer une étude par fichier PGN (3 études au total)
- Noms suggérés : "Blancs - Gambit Dame", "Noirs vs e4 - Scandinave", "Noirs vs d4 - Slave"

## Importer un PGN dans une étude

### Étapes
1. Dans l'étude, cliquer sur "+" pour ajouter un chapitre [À CONFIRMER]
2. Aller dans l'onglet "PGN" [À CONFIRMER]
3. Coller le contenu du fichier PGN ou importer le fichier [À CONFIRMER — coller ou upload ?]
4. Cliquer sur "Create chapter" / "Créer le chapitre" [À CONFIRMER]
5. Chaque `[Event "..."]` du PGN devrait créer un chapitre séparé [À CONFIRMER — ou faut-il importer chapitre par chapitre ?]
